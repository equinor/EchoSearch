# Install

### NPM

```sh-session
npm install @equinor/echo-search --save
```

# Development

## Run

For development we have provided a test playground to be able to test searchs. This can be run by using.

```sh-session
npm start
```

## Test

Echo search is set upt to run Jest test suite. To run tests use.

```sh-session
npm run test
```

For abetter overview, run tests with test coverage.

```sh-session
npm run test-coverage
```
